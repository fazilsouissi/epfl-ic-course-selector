## ‚ú® Features

- **Interactive Course Planning**: Drag and drop courses into your semester plan (BA3-BA6).
- **Smart Filtering**: search by name, abbreviation, or category.
- **Credits Tracking**: Automatically calculates credits per semester and block.
- **Shareable Links**: Share your course selection with a compressed URL.
- **Import/Export**: Save your configuration as JSON or load it later.
- **Dark Mode**: Fully supported dark/light theme.
